# STOCK-ADJUST-REJECT-NO-EXPIRY-001

## 状态

已确认。

## 业务规则

- 已驳回的人工库存调整申请长期保留，不设置自动关闭、自动过期或自动删除时间。
- 业务员可以随时从“我的申请”进入原驳回申请进行修改。
- 修改后必须在原申请上重新提交审核，不新建一张申请替代原申请。
- 业务员可以修改整张申请，包括新增或删除商品、修改各商品调整数量、调整原因和备注。
- 原驳回理由必须保留，不能因修改或重新提交被覆盖、删除。
- 申请、提交、驳回、修改和重新提交等历史操作记录必须完整保留。
- 申请处于已驳回或修改待提交状态时，不得改变正式库存，也不得生成正式库存流水。

## 实施约束

- 不得通过定时任务将已驳回申请自动转为关闭、失效或删除状态。
- 重新提交必须沿用原申请身份，并新增版本或操作历史；历史审核结果和驳回理由只读保留。
- 已驳回申请长期未修改时仍保持可进入、可编辑、可重新提交。
